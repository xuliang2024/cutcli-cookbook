# Showcase

[English](README.md) · [简体中文](README.zh.md)

Each month we feature community work on cutcli.com's homepage banner, Twitter, and RedBook.

## How to submit

Open an issue with the [Showcase issue template](https://github.com/xuliang2024/cutcli-cookbook/issues/new?template=showcase.yml) and include:

- Title of your work
- Link to the work (video / article / Douyin / Bilibili / Twitter all welcome)
- Author byline + link
- Process notes (which cutcli commands? which case did you build on?)
- Agreement to CC-BY 4.0 for the screenshot/cover (you keep the original work's copyright)

## Selection criteria

Featured monthly on the 1st, 5-10 entries chosen by:

1. **Novelty**: did something with cutcli that nobody had thought of
2. **Completeness**: a finished piece, not a demo
3. **Reproducibility**: you can write a short "here's how" and share it with the community
4. **Licensing**: assets are CC0 / yours / properly licensed
