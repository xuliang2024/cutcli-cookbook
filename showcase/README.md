# Showcase

每月精选社区作品，发到 cutcli.com 首页 banner、Twitter、小红书。

## 怎么投稿

按 [Showcase Issue 模板](https://github.com/m007/cutcli-cookbook/issues/new?template=showcase.yml) 开一个 issue，填上：

- 作品标题
- 作品链接（视频/文章/抖音/B站/Twitter 都行）
- 作者署名 + 链接
- 制作过程简介（用了哪些 cutcli 命令？参考了哪个 case？）
- 同意 CC-BY 4.0 授权（截图/封面，原作品版权仍归你）

## 评选规则

每月 1 号公布上月精选 5-10 个，标准：

1. **新颖**：用 cutcli 做了别人没想到的事
2. **完整**：作品本身完成度高，不是 demo
3. **可复现**：你能写一段简单的"我是这么做的"分享给社区
4. **遵守版权**：素材 CC0 / 你拥有版权 / 已获授权
